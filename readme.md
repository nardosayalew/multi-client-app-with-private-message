# Java Socket Chat Application

An **expert-level Java Socket Chat Application** built using **Java Sockets, Multithreading, and Swing GUI**.

This project demonstrates real-time communication between multiple users with authentication and a clean user interface.

---

##  Features

-  Login & Password Authentication
-  Public Chat (Broadcast)
-  Private Messaging
-  Online Users List
-  Chat History (Server-side)
-  Simple & Clean Swing GUI
-  Multi-client, Multi-threaded Server
-  Runs on Localhost
